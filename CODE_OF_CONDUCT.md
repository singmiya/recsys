# 行为准则 (Code of Conduct)

## 我们的承诺

为了建立和维护一个开放、欢迎和安全的社区，我们、作为贡献者和维护者，承诺：

- 尊重所有参与者，无论其性别、性别认同、性别表现、性取向、年龄、身体残疾、残疾状态、种族、民族、种族血统、宗教、社会经济地位、教育水平、国籍或经验水平如何
- 创造一个欢迎、包容和友好的环境
- 接受建设性的批评，专注于改进项目
- 关注对社区最有益的东西
- 对彼此表现出同情和尊重

## 我们的标准

有助于创造积极环境的行为包括：

- 使用包容性语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最好的事情
- 表现出同情和尊重

有助于创造消极环境的行为包括：

- 使用贬低或歧视性的语言或图像
- 人身攻击
- 骚扰或霸凌
- 不欢迎的性关注
- 发布他人的私人信息
- 其他我们认为不适当的行为

## 执行

行为准则违规可以通过以下方式报告：

**电子邮件**：通过 GitHub Issues 或直接发送电子邮件至项目维护者

维护者有责任：
- 接收和审查报告
- 进行调查
- 采取适当的行动
- 为受害者提供支持

## 后果

不符合行为准则的参与者可能会面临以下后果：

1. **警告** - 首次轻微违规
2. **暂时禁用** - 严重或反复违规
3. **永久禁用** - 极端情况

## 适用范围

此行为准则适用于：
- 项目空间（Issue、PR、Discussion）
- 官方项目事件
- 与项目相关的线上线下交互

## 参考

本行为准则基于 [Contributor Covenant](https://www.contributor-covenant.org/)，版本 2.1
